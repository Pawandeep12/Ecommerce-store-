# Mini E-Commerce Store

## Overview
This is a simple E-Commerce Store built with HTML, CSS, and JavaScript.  
Users can browse products, add them to the shopping cart, update quantities, remove items, and see the total price.  
The cart persists using localStorage, making the app more realistic.

---

## Features
- Display a list of products with name and price  
- Add products to cart  
- Increase or decrease product quantity in cart  
- Remove products from cart  
- Real-time total price calculation  
- Cart data persists in localStorage 
- Clean and responsive UI  

---

## Technologies Used
- HTML
- JavaScript (ES6+)
- DOM Manipulation
- LocalStorage for data persistence
- CSS
